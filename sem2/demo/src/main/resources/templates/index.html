Информационная система «Гостиница»

Данное приложение разработано в рамках курсового проекта по дисциплине «Разработка кросс-платформенных приложений» с использованием фреймворка Spring.

Описание

Приложение предназначено для автоматизации работы гостиницы. Оно позволяет управлять информацией о номерах, клиентах и бронированиях.
